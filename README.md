# MoodTunes 🎵 

MoodTunes is an engaging web app that helps users track their daily moods through music. Each day, users select their emotional state and receive a perfectly matched song recommendation. The app creates beautiful visualizations of their mood patterns and builds personalized playlists.

Key Features:
- Interactive mood selection through an intuitive emoji wheel
- Daily song recommendations based on mood
- Beautiful mood tracking visualizations
- Shareable mood+song cards for social media
- Progressive Web App support

Tech Stack:
- HTML5, CSS3, JavaScript
- Web Audio API for music previews
- LocalStorage for offline support
- Chart.js for visualizations
